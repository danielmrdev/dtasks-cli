  <behavior>
    - `dtasks update` with up-to-date binary: prints "Already up to date (v0.3.0)" or similar
    - `dtasks update --json` with up-to-date binary: outputs {"current":"0.3.0","latest":"0.3.0","updated":false,"message":"already up to date"}
    - `dtasks update --json` when update available: downloads binary, outputs {"current":"0.2.0","latest":"0.3.0","updated":true,"message":"updated to v0.3.0"}
    - Version comparison: "v"+current == latest OR current == latest — handles both "v0.3.0" and "0.3.0" ldflags formats
    - On download error: returns error (non-zero exit), JSON: {"current":"...","latest":"...","updated":false,"message":"update failed: ..."}
    - After successful update: calls skill.PromptAndInstall with os.Stdin, os.Stdout and embedded skill content; errors from skill install are printed as warnings, not fatal errors
    - Post-update prints hint: "Run install.sh to update shell completions" (COMP-04)
  </behavior>
  <action>
    Create cmd/update.go:

    1. Embed the skill file at the top of the file:
       ```go
       //go:embed ../skills/dtasks-cli/SKILL.md
       var embeddedSkill []byte
       ```

    2. Define UpdateResult struct:
       ```go
       type UpdateResult struct {
           Current string `json:"current"`
           Latest  string `json:"latest"`
           Updated bool   `json:"updated"`
           Message string `json:"message,omitempty"`
       }
       ```

    3. updateCmd.RunE logic:
       a. current = rootCmd.Version (may be "v0.3.0" or "0.3.0" — normalize: strings.TrimPrefix(current, "v"))
       b. Call updater.FetchLatestVersion("danielmrdev/dtasks-cli") — on error return error
       c. latest = strings.TrimPrefix(latestTag, "v")
       d. If current == latest: emit result with updated=false, message="already up to date"
       e. Otherwise: call updater.AssetName() then updater.DownloadAndReplace(url, exePath)
          - exePath: os.Executable() — error if fails
          - assetURL: fmt.Sprintf("https://github.com/danielmrdev/dtasks-cli/releases/download/v%s/%s", latest, asset)
          - On replace error: return error
       f. After successful update: call skill.PromptAndInstall(homeDir, embeddedSkill, os.Stdin, os.Stdout); if err, fmt.Fprintln(os.Stderr, "warning: skill install:", err) (non-fatal)
       g. Print post-update hint to stdout: "Run install.sh to update shell completions"
       h. Emit result with updated=true

    4. JSON output: if output.JSONMode, use json.NewEncoder(os.Stdout).Encode(result); otherwise fmt.Println the message
